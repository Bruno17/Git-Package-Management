Define package dependencies here.

**Wrapper:** dependencies (array)
```json
{
    "dependencies": []
}
```

#### Available properties:
* **name** (required) - Name of the dependent package
* **version** (required) - Version of the package

#### Example
```json
{
    "dependencies": [{
        "name": "collections",        
        "version": ">=3.2.0"
    }]
}
```
